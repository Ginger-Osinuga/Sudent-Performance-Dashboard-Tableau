# 🎓 Student Performance Analysis

### Project Overview
This project analyzes student academic performance across core subjects (Math, Reading, and Writing) to identify demographic trends and performance gaps. Using **Tableau**, the project provides an interactive way to visualize how gender impacts subject-specific outcomes, allowing educators to pinpoint where students excel and where they may need additional support.

### 🔍 Methodology
*   **Data Organization:** Structured student test data into three primary performance categories: Math, Reading, and Writing.
*   **Analysis:** Segmented the data by gender to uncover hidden disparities that general school averages often mask.
*   **Visualization:** Built an interactive Tableau dashboard featuring:
    *   **Subject Averages:** A high-level view of curriculum performance.
    *   **Demographic Filtering:** Real-time filtering to compare male vs. female performance trends.
    *   **Gap Identification:** Highlighted the specific "Math Gap" and "Literacy Advantage" across groups.

### Key Findings & Strategic Recommendations
*   **The Math Performance Gap:** A significant disparity exists in STEM subjects; male students (68.7) outperform female students (63.6) by approximately 5.1 points.
    *   **Strategic Recommendation:** Implement targeted Math enrichment and mentorship programs specifically for female students to bridge this 8% performance gap.
*   **Literacy Strengths:** Female students demonstrate a commanding lead in literacy, with an average score of 73 in Reading—the highest single metric in the dataset.
    *   **Strategic Recommendation:** Leverage high-performing female students as peer tutors in writing workshops to help improve male literacy scores, which currently lag at 63.
*   **Subject Balance:** While the overall school average remains consistent (66–69), interactivity reveals that these numbers are heavily influenced by gender-specific strengths.
    *   **Strategic Recommendation:** Move away from "one-size-fits-all" curriculum adjustments and instead focus on gender-specific workshops for Math (Females) and Writing (Males).

### Data Source
*   **Dataset:** Student Performance in Exams (Kaggle/)
*   **Key Metrics:** Gender, Math Score, Reading Score, Writing Score.
